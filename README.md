OBJETIVO DEL PROYECTO DE ACUERDO CON LOS INDICADORES DE EVALUACIÓN
- Permitir al usuario añadir, ver, editar y eliminar tareas mediante formularios.
- Sincronizar las tareas con una API REST (GET, POST, PUT, DELETE).
- Mantener el estado global de la aplicación con Context API o Redux.
- Utilizar Expo Router para manejar múltiples pantallas y pasar parámetros entre ellas.
- Implementar tipado con TypeScript en componentes y datos.
